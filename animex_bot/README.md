# 🤖 ربات لیچر انیمکس (AnimeX Leecher Bot)

ربات تلگرام حرفه‌ای برای استخراج خودکار **لینک‌های دانلود** از سایت [animex.click](https://animex.click/) — تک‌فایل PHP، بدون نیاز به پروسس دائمی (وب‌هوک)، مناسب **هاست اشتراکی** (cPanel / DirectAdmin).

---

## ✨ امکانات

- 🎬 **/start** و خوش‌آمدگویی + پنل کاربری داخل ربات
- 🔗 فرستادن لینک انیمه/فیلم/سریال از animex.click:
  - 🖼️ ارسال **پوستر** با کپشن (عنوان، سال، ژانر، امتیاز، وضعیت، کشور، رده سنی، توضیحات)
  - دکمه **«🎬 مشاهده لینک‌های دانلود»** → ساخت فایل مرتب `.txt` شامل **همه لینک‌ها**
- 📁 دسته‌بندی هوشمند لینک‌ها: **فصل/نسخه → نوع زیرنویس (سافت‌ساب/دوبله/زیرنویس چسبیده) → کیفیت (4k/1080p/...) + سایز**
- 🧩 باز کردن پوشه‌های دانلود (`?dir=`) و آوردن لینک تک‌قسمت‌ها (در صورت امکان)
- 📊 محدودیت روزانه **۵۰۰ درخواست** برای کاربر عادی (قابل تغییر با `/setlimit`)
- 💎 کاربر **VIP** بدون محدودیت تا تاریخ انقضا
- 🛡️ **پنل ادمین وب** + دستورات ادمین داخل ربات
- 🔒 امن: توکن داخل `config.php` (با باز شدن از وب خروجی نمی‌دهد) + `.htaccess` محافظت از دیتابیس
- ⚡ کش نتایج برای سرعت و کاهش بار

---

## 📋 پیش‌نیازها (در اکثر هاست‌ها فعّال است)

- PHP **7.4 یا بالاتر**
- افزونه‌ها: `curl` , `sqlite3` , `mbstring` , `dom` (xml)
- گواهی **SSL/HTTPS** فعال برای دامنه (وب‌هوک تلگرام فقط روی HTTPS کار می‌کند — AutoSSL رایگان cPanel کافی است)

## 🚀 نصب — روش سریع (تک‌خطی، از طریق SSH / ترمینال cPanel)

اگر به هاست SSH دارید یا در cPanel بخش **Terminal** باز است، این دستور را اجرا کنید:

```bash
mkdir -p ~/public_html/bot && cd ~/public_html/bot && \
B=https://raw.githubusercontent.com/Novoo99/NVpn/arena/019f90d8-nvpn/animex_bot && \
wget -q "$B/animex_bot.php" && wget -q "$B/jdatetime.php" && wget -q "$B/.htaccess" && \
echo "✅ فایل‌ها نصب شدند"
```

> دقت: آدرس پوشه را اگر می‌خواهید تغییر دهید (`bot` را در `mkdir` و آدرس باز کردن عوض کنید).

## 📂 نصب — روش دستی (cPanel ▸ File Manager)

1. از [@BotFather](https://t.me/BotFather) یک ربات بسازید و **توکن** را بردارید.
2. وارد cPanel ▸ **File Manager** ▸ `public_html` شوید.
3. یک پوشه مثلاً `bot` بسازید و این سه فایل را در آن آپلود کنید:
   - `animex_bot.php`
   - `jdatetime.php`
   - `.htaccess`

## ⚙️ راه‌اندازی (۱۵ ثانیه)

آدرس زیر را در مرورگر باز کنید (توکن خود را جایگزین کنید):

```
https://دامنه-شما/bot/animex_bot.php?setup=توکن-ربات-شما
```

صفحه‌ای باز می‌شود که:
- ✅ توکن را اعتبارسنجی می‌کند
- ✅ وب‌هوک را روی تلگرام تنظیم می‌کند
- ✅ دیتابیس را می‌سازد
- 🔑 کد مالکیت (`/claim XXXX`) را نشان می‌دهد
- 🖥️ لینک **پنل ادمین وب** را می‌دهد

سپس در تلگرام ربات را `/start` کنید و این پیام را بفرستید تا **ادمین ارشد** شوید:

```
/claim کدی-که-در-صفحه-نمایش-داده-شد
```

> توجه: طبق درخواست شما، **هیچ‌کس در ابتدا ادمین نیست**. فقط کسی که کد `/claim` را دارد ادمین می‌شود.

## 🧪 تست استخراج (بدون تلگرام)

برای دیدن خروجی استخراج به‌صورت JSON در مرورگر:

```
https://دامنه-شما/bot/animex_bot.php?test=https://animex.click/anime/solo-leveling-season-2-arise-from-the-shadow/
```

اگر همه‌چیز درست باشد، لیست گروه‌بندی‌شده لینک‌ها را در JSON می‌بینید.

## 🎛️ دستورات

### 👤 کاربر
| دستور | کاربرد |
|------|--------|
| `/start` | خوش‌آمدگویی |
| `/panel` یا `/myaccount` | پنل کاربری (مصرف امروز/VIP) |
| `/id` | شناسه عددی شما |
| `/help` | راهنما |
| *(فرستادن لینک)* | استخراج اطلاعات و لینک‌ها |

### 🛡️ ادمین (تنها ادمین‌ها)
| دستور | کاربرد |
|------|--------|
| `/setvip <id> <days>` | دادن VIP به کاربر برای N روز |
| `/ban <id> [دلیل]` | مسدود کردن کاربر |
| `/unban <id>` | رفع مسدودیت |
| `/addadmin <id>` | ادمین کردن |
| `/deladmin <id>` | گرفتن ادمینی |
| `/userinfo <id>` | اطلاعات کاربر |
| `/setlimit <n>` | تغییر سقف روزانه (پیش‌فرض ۵۰۰) |
| `/stats` | آمار ربات |
| `/broadcast <متن>` | پیام همگانی |

## 🧯 عیب‌یابی

- **«وب‌هوک تنظیم نشد»**: مطمئن شوید دامنه روی **HTTPS** است و آدرس از بیرون قابل دسترسی است.
- **توکن نشت نکرده؟** خیر؛ در `config.php` ذخیره می‌شود که از وب خروجی نمی‌دهد. دیتابیس و `.json` هم با `.htaccess` بسته شده‌اند.
- **صفحه نصب سفید/خطا**: PHP ورژن و افزونه `sqlite3`/`curl` را در cPanel چک کنید.
- **استخراج خالی/ناقص**: با `?test=لینک` خروجی را ببینید و در صورت نیاز گزارش دهید تا الگو بهبود یابد.
- **حذف وب‌هوک** (برای دیباگ): `https://دامنه/bot/animex_bot.php?unhook=توکن`
